Projets Réalisés

Projets SAE

1. Site Web de Géolocalisation et de Suivi de Parcours

  • Objectif : Suivre en temps réel les déplacements sur une carte interactive.
  • Détails :
    • Interface intuitive pour la consultation et l'enregistrement des parcours.
    • Intégration de systèmes GPS pour capturer les trajets en direct.
    • Cartographie dynamique avec OpenLayers ou Leaflet.

2. Logiciel de Gestion des Données Cartographiques

  • Objectif : Visualiser et analyser les accidents de vélo à Montpellier.
  • Détails :
    • Collecte et structuration des données dans une base relationnelle.
    • Visualisation des incidents sur une carte interactive avec filtres.
    • Analyse des données pour identifier les zones à risque.

Projets Personnels

1. Site Web d'Itinéraires Souvenirs

  • Objectif : Créer des itinéraires personnalisés enrichis de photos et souvenirs.
  • Détails :
    • Ajout de points d'intérêt avec descriptions et photos.
    • Partage des itinéraires avec d'autres utilisateurs.
    • Utilisation d'API cartographiques comme Google Maps.

2. Mini Stations Météo Connectées

  • Objectif : Surveiller les conditions météorologiques en temps réel.
  • Détails :
    • Capteurs pour mesurer température, humidité, pression.
    • API RESTful pour transmettre les données à un serveur.
    • Affichage des données sur une carte interactive.
